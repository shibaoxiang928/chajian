# 快速开始指南

## 🎯 推荐方案（按简单程度排序）

### ⭐ 方案一：Auto.js脚本（最简单，强烈推荐！）

**优点：**
- ✅ 无需电脑，直接在手机上运行
- ✅ 无需编译，无需root
- ✅ 易于修改和调试

**步骤：**
1. 在手机上安装 **Auto.js** 应用
2. 将 `Auto.js脚本版本/autoplay.js` 导入Auto.js
3. 开启Auto.js的无障碍权限
4. 运行脚本即可

**详细说明：** 查看 [Auto.js脚本版本/README.md](Auto.js脚本版本/README.md)

---

### 方案二：GitHub Actions在线编译

**优点：**
- ✅ 无需本地安装任何工具
- ✅ 自动构建并下载APK
- ✅ 完全免费

**步骤：**
1. 将项目上传到GitHub
2. 在GitHub仓库页面，点击"Actions"
3. 选择"Build APK"工作流，点击"Run workflow"
4. 构建完成后下载APK
5. 安装到手机并启用无障碍服务

**详细说明：** 查看 [编译指南-命令行.md](编译指南-命令行.md) 的方案二

---

### 方案三：命令行编译

**适合：** 已有Android SDK环境

**步骤：**
```bash
# Windows
.\gradlew.bat assembleDebug

# Linux/Mac  
./gradlew assembleDebug

# 安装
adb install app/build/outputs/apk/debug/app-debug.apk
```

**详细说明：** 查看 [编译指南-命令行.md](编译指南-命令行.md) 的方案一

---

### 方案四：Frida脚本（需要root）

**适合：** 开发和调试

**步骤：**
1. 安装frida-tools: `pip install frida-tools`
2. 在手机上安装frida-server
3. 运行: `frida -U <包名> -l frida脚本版本/autoplay.js`

**详细说明：** 查看 [frida脚本版本/README.md](frida脚本版本/README.md)

## 📱 使用流程（所有方案通用）

1. **启用无障碍服务**
   - 打开系统设置 → 无障碍
   - 找到对应的服务并开启

2. **打开学习应用**
   - 启动科目一学习应用
   - 开始播放视频

3. **自动工作**
   - 视频播放完成后，插件会自动检测弹窗
   - 自动点击"开始学习"按钮
   - 自动切换到下一个视频

4. **停止脚本**
   - Auto.js: 按音量下键
   - APK版本: 在无障碍设置中关闭服务
   - Frida: Ctrl+C

## 🔧 自定义配置

如果目标应用的按钮文本不同，需要修改：

**Auto.js版本：**
```javascript
// Auto.js脚本版本/autoplay.js
const CONFIG = {
    BUTTON_TEXT: "开始学习",  // 修改这里
    ...
};
```

**APK版本：**
```java
// app/src/main/java/com/xuexi/autoplay/AutoPlayService.java
private static final String BUTTON_TEXT_START_LEARNING = "开始学习"; // 修改这里
```

**Frida版本：**
```javascript
// frida脚本版本/autoplay.js
const BUTTON_TEXT = "开始学习"; // 修改这里
```

## ❓ 常见问题

### Q: 哪个方案最简单？
**A:** Auto.js脚本版本最简单，推荐使用！

### Q: 没有root能用吗？
**A:** 可以！Auto.js和APK版本都不需要root。

### Q: 需要电脑吗？
**A:** Auto.js版本不需要，直接在手机上运行即可。

### Q: 如何知道应用的包名？
**A:** 
- 使用 `adb shell pm list packages` 查看所有包名
- 或使用应用信息查看工具

### Q: 脚本无法找到按钮？
**A:** 
1. 检查按钮文本是否正确
2. 使用Auto.js的"布局分析"功能查看实际UI
3. 调整查找条件
